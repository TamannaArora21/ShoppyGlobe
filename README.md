# ShoppyGlobe - E-Commerce React Application

## Project Overview

ShoppyGlobe is a React-based e-commerce application that allows users to browse products, search products, view product details, add items to a shopping cart, manage quantities, and complete a checkout process.

The project is built using React, React Router, Redux Toolkit, and React Redux. Product data is fetched from the DummyJSON API.

---

## Features

* Product Listing
* Product Search
* Product Detail Page
* Add to Cart
* Remove from Cart
* Increase/Decrease Quantity
* Cart Item Counter
* Checkout Page
* Clear Cart After Order Placement
* 404 Not Found Page
* API Integration using Custom Hooks
* Centralized State Management using Redux Toolkit

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start development server

```bash
npm run dev
```
